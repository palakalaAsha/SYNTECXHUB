# Expense Tracker CLI

## Project Overview

Expense Tracker CLI is a Python-based command-line application that helps users manage their daily income and expenses. It allows users to record transactions, view all records, search by category, generate monthly summaries, and visualize expenses using charts.

## Features

- Add Income
- Add Expense
- View All Transactions
- Search Transactions by Category
- Monthly Summary
- Generate Pie Chart
- Generate Bar Chart
- Store Data in CSV File

## Technologies Used

- Python
- CSV
- Datetime
- Matplotlib

## How to Run

1. Install Python.
2. Install required library:

pip install -r requirements.txt

3. Run the project:

python main.py
